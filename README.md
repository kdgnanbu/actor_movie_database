# actor_movie_database

## 概要
俳優と映画２つの情報を利用して表示するWebアプリケーションです。 
データをDBに保存されているデータを使い表示をします。

---

## 使用技術
- HTML
- CSS
- JavaScript
- Vue
- node
- MySQL

---

## アプリ内容・機能
- 俳優一覧表示機能
- 映画一覧表示機能
- 検索機能

---

## 工夫した点
- DBの情報を最低限にして、SQLを使い必要な情報を付け足してnodeに保存しました
- コンポーネントを意識して、少ないコード量で同じ動きができるようにしました
